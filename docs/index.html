Getting Started

NextGen CSS Framework is a lightweight framework that helps you build modern websites quickly and efficiently. It provides a flexible grid system, various components, and utility classes.

Installation

There are several ways to include NextGen CSS Framework in your project:

NPM
npm install nextgen-css-framework --save
CDN
<link rel="stylesheet" href="https://cdn.example.com/nextgen/dist/nextgen.min.css">
Direct Download

Download the CSS files from our repository and include them in your project:

<link rel="stylesheet" href="path/to/nextgen.css">

Grid System

NextGen CSS Framework includes a powerful flexbox-based grid system that helps you create responsive layouts with ease. The grid is based on a 12-column layout.

Basic Grid

.col-md-4
.col-md-4
.col-md-4

  <div class="col-md-4">.col-md-4</div>
  <div class="col-md-4">.col-md-4</div>

</div>

Dark Mode Support

NextGen CSS Framework includes automatic dark mode support using CSS variables and the prefers-color-scheme media query.

Dark Mode Compatible

This card will automatically adapt to dark mode when the user's system preference is set to dark.
